# love_kb

Персональная вики о том, **что наука знает о любви** — природа привязанности, контактный
комфорт, материнская депривация, теория привязанности, нейробиология и психология любви.
Построена по паттерну incremental-вики (Karpathy `llm-wiki.md`): знание компилируется в связный
набор markdown-страниц со ссылками `[[вики-ссылки]]`, а не выводится заново на каждый запрос.

Сестринская база — `../bania_kb` (та же механика, другой домен). Отличие love_kb — **научная**
дисциплина источников: популярные пересказы сверяются с первичной литературой, расхождения
фиксируются явно (раздел «Проверка фактов» на каждой популярной source-странице).

## Что внутри

- `wiki/topics/` — синтез по темам (тезисы с атрибуцией, противоречия, открытые вопросы).
- `wiki/concepts/` — научные понятия, эффекты, теории, термины.
- `wiki/entities/` — учёные, институции, лаборатории, аппараты, движения.
- `wiki/sources/` — страница на каждый источник (TL;DR, тезисы, проверка фактов, связи).

Первичные источники (полные тексты статей, PDF научных работ) в публичный репозиторий не входят —
публикуется только вики.

## Сайт (SPA)

Статический клиентский SPA, раздаётся через GitHub Pages. Без бэкенда и сборочного тулчейна на
стороне Pages — всё предсобрано.

- `build_site.py` — генератор: парсит `wiki/`, разрешает `[[wikilinks]]` в `#/p/<slug>`, считает
  обратные ссылки, пишет `data/pages.json`.
- `index.html` + `assets/` — оболочка, рендер (marked), поиск (MiniSearch), навигация, backlinks,
  тёмная/светлая тема.

Пересборка после правок вики:

```bash
python3 build_site.py           # → data/pages.json
python3 build_site.py --check   # + отчёт по orphan-ссылкам
```

Локальный просмотр:

```bash
python3 -m http.server 8000     # затем http://localhost:8000/
```

## Health-check

```bash
python3 ../love_lint.py         # orphans / broken / dups / stubs / isolated / unverified
```

`unverified` — love-специфичная секция: популярные source-страницы без раздела «Проверка фактов».

## Старт

Первая тема — **Гарри Харлоу и «природа любви»**: surrogate-mother эксперименты, контактный
комфорт, депривация, «колодец отчаяния». Повод — популярная статья «Эксперимент, который запретили
навсегда»; от неё — к первичным публикациям Харлоу и теории привязанности.
