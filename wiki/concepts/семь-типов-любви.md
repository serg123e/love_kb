# Семь типов любви (Стернберг)

Комбинации трёх компонентов [[треугольная-теория-любви|треугольной теории]] ([[роберт-стернберг| Стернберг]]) дают 8 состояний; 7 — виды любви, 8-е — не-любовь (нет ни одного компонента).

| Тип | Близость | Страсть | Обязательство |
|---|:---:|:---:|:---:|
| Симпатия (liking) | ✓ | — | — |
| Влюблённость-увлечение (infatuation) | — | ✓ | — |
| Пустая любовь (empty) | — | — | ✓ |
| Романтическая (romantic) | ✓ | ✓ | — |
| Компаньонская (companionate) | ✓ | — | ✓ |
| Безрассудная (fatuous) | — | ✓ | ✓ |
| **Совершенная (consummate)** | ✓ | ✓ | ✓ |

## Тезисы

- **Компаньонская** (близость+обязательство, без страсти) — частый исход долгого брака; смыкается с
  [[привязанность-у-взрослых|привязанностью]].
- **Безрассудная** (страсть+обязательство без близости) — «вихрь» помолвки без эмоциональной связи.
- ⚠️ Типы — эвристические зоны континуума, не строгие категории (см. [[треугольная-теория-любви]]).
