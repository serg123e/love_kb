# Русскоязычная литература: совместимость, удовлетворённость браком, подбор пар

Подборка русскоязычных научных публикаций по теме [[подбор-пар]]. Приоритет — открытый доступ
(КиберЛенинка, voppsy.ru, psyjournals.ru). Выходные данные триангулированы; где не сверено с
первоисточником — пометка ⚠️. **Главный вывод подборки** — в конце (рус-традиция почти не знает
западного скепсиса о предсказуемости влечения).

## 1. Психологическая совместимость супругов

- **Обозов Н.Н., Обозова А.Н. (1981).** Три подхода к исследованию психологической совместимости. //
  Вопросы психологии, №6: 98–101. — классическое разведение **«совместимости» и «срабатываемости»**;
  структурный/функциональный/адаптивный подходы. *(Открытый текст voppsy.ru.)*
- **Обозов Н.Н., Обозова А.Н. (1982).** Диагностика супружеских затруднений. // Психологический журнал,
  3(2): 147–151 ⚠️ (страницы). — индикаторы устойчивости брака: совместимость и срабатываемость.
- **Обозов Н.Н. (1990).** Психология межличностных отношений. — Киев: Лыбидь, 191 с. — монография о
  типах личности и (не)совместимости. **Уровневая модель** (духовная/психологическая/семейно-ролевая/
  физиологическая совместимость).

## 2. Удовлетворённость браком и ролевые ожидания (методики)

- **Столин В.В., Романова Т.Л., Бутенко Г.П. (1984).** Опросник удовлетворённости браком (**ОУБ**). //
  Вестник МГУ. Сер.14. Психология, №2: 54–61. — экспресс-методика субъективной удовлетворённости.
- **Волкова А.Н., Трапезникова Т.М. (1985).** Методические приёмы диагностики супружеских отношений. //
  Вопросы психологии, №5: 110–116. — первоисточник методики **РОП «Ролевые ожидания и притязания в
  браке»** (согласованность ролей). *(Открытый текст.)* См. [[фундаментальная-совместимость-целей]].
- **Алёшина Ю.Е. (1987).** Цикл развития семьи: исследования и проблемы. // Вестник МГУ. Сер.14, №2:
  60–72. — удовлетворённость браком по стадиям жизненного цикла семьи.

## 3. Привязанность взрослых (адаптации западных инструментов)

- **Казанцева Т.В. (2008).** Адаптация методики «Опыт близких отношений» (К. Бреннан, Р.К. Фрейли). //
  Известия РГПУ им. Герцена ⚠️. — русская версия **ECR-R** (шкалы тревога/избегание). Связь:
  [[двумерная-модель-привязанности]].
- **Сабельникова Н.В., Каширский Д.В. (2015).** Опросник привязанности к близким людям. //
  Психологический журнал, 36(4): 84–97. — психометрический инструмент на базе **ECR** на росс. выборке.
- **Сабельникова Н.В., Каширский Д.В., Нефедова А.В. (2024).** Русскоязычная версия опросника **ECR-RS**
  (Fraley). // Психологический журнал, 45(4): 97–113. — контекст-специфичная привязанность (партнёр/
  мать/отец/друг). См. [[adult-attachment-interview]], [[fraley-waller-brennan-2000]].
- **Екимчик О.А. (2011).** Адаптация «Треугольной шкалы любви» Р. Стернберга. // Вестник Костромского
  ГУ, 17(4): 216–221 ⚠️. — рус. адаптация (близость/страсть/обязательства). См.
  [[треугольная-теория-любви]]. *(И 2015 — «Шкала историй любви» Стернберга.)*

## 4. Стабильность брака, факторы развода

- **Сысенко В.А. (1981).** Устойчивость брака: проблемы, факторы, условия. — М.: Финансы и статистика. —
  классическая монография о факторах устойчивости и конфликтах. *(+ «Супружеские конфликты», 1983/1989.)*
- **Левченко М.А., Адушкина К.В. (2017).** Факторы удовлетворённости браком на разных этапах ЖЦ семьи. //
  Бюллетень науки и практики, №9(22): 243–249. — эмпирика по стадиям. *(КиберЛенинка.)*
- **Сычев О.А., Аношкин И.В. (2024).** Русскоязычные версии опросников стилей и эффективности разрешения
  конфликтов. // Психология. Журнал ВШЭ, 21(1): 32–54. — методики **по Готтману** (N=415);
  «непродуктивный паттерн конфликта» как предиктор нестабильности. См. [[четыре-всадника]]. *(PDF.)*
- **Лагойда Н.Г. (2017).** Проблема стабильности брака и роста числа разводов. // Вестник Бурятского ГУ,
  №5: 86–96 ⚠️. — анализ факторов разводимости.

## 6. Онлайн-знакомства (социология рунета)

  практика общественного развития, №7: 97–106. — **единственная** рус. работа, прямо опирающаяся на

## Чем рус-традиция отличается от западной

1. **Другой объект.** Запад ([[finkel-2012-online-dating|Finkel]], [[joel-2017-romantic-desire|Joel]])
   критикует обещание «алгоритм предскажет вашу пару» и показывает невыводимость
   [[предсказуемость-совместимости|relationship-эффекта]]. Рус-традиция — про **совместимость уже
   сложившейся пары** и её связь с удовлетворённостью браком (уровневая модель Обозова, ролевые
   ожидания Волковой). Это пост-фактум диагностика, а не предиктивный отбор незнакомцев.
2. **Центр тяжести методик** — ОУБ (Столин), РОП (Волкова), совместимость по Обозову. Привязанность
   взрослых пришла позже и как **адаптации** западных инструментов (ECR/ECR-RS, шкалы Стернберга).
3. **Онлайн-дейтинг изучается социологически** (рынок/базар, самопрезентация), а не
   психометрически-предиктивно.

## ⚠️ Главный пробел (важно для базы)

**Русской науки о (не)предсказуемости влечения и о критике «алгоритмов совместимости» практически
нет:** [[joel-2017-romantic-desire|Joel 2017]] — ноль русских цитат, [[finkel-2012-online-dating| Finkel]] — единичные. Коммерческие «научные подборы» (брачные агентства, астро/нумерологические
«калькуляторы совместимости») независимой научной проверки не имеют. То есть ключевой тезис
[[методика-подбора-пар|методики]] (совместимость не предсказать по анкете) в рунете **не освещён** —
это и угроза (рынок полон необоснованных обещаний), и ниша.
