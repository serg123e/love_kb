# Eastwick et al. (2014) — предиктивная валидность идеалов партнёра (мета-анализ)

Мета-анализ ([[пол-иствик|Иствик]] и др.; Psychological Bulletin 140(3):623–665, **k=97**).

## Ключевые результаты

- Внешность предсказывает романтическую оценку с эффектом **r ≈ .40** (у обоих полов); доход —
  **r ≈ .10** (у обоих). Половые различия в *корреляциях* ничтожны (r_diff = .03, незначимы).
- **Главное:** соответствие (matching) идеалов человека реальным чертам партнёра **слабо предсказывает
  исходы**, особенно в живых/устоявшихся отношениях. Идеалы работают для *гипотетических* мишеней, не
  для реальных людей, которых уже встретил.

## Значение для подбора

**Preference-matching** (подбор по совпадению «идеала» с профилем) имеет низкую предиктивную
валидность. См. [[предсказуемость-совместимости]].
